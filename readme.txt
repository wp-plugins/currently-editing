Quickly see who is currently editing a post or page without navigating to that post/post.

== Description ==

Generates a visual representation of which posts/pages are currently being edited in your dashboard. Saves time by not having to navigate to each page/post individually. Great for large sites with multiple authors and admins such as magazine sites or networks.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= What does this plugin do? =

Simply adds a visual representation from the built in timestamp and shows you who is editing which pages/posts.

= Does it work with custom post types? =

Yes, it does.
